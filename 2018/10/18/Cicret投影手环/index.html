文章参考：工程师解读Cicret投影手环 就是个忽悠！
##部件图

  • accelerometer：加速度计
  • memory card + rom：存储卡和只读存储器
  • processor：处理器
  • vibrator：震动器
  • micro USB port：微型USB接口
  • battery：电池
  • proximity sensor long range：长距离传感器
  • pico-projector：微型投影仪
  • bluet low energy：低功耗蓝牙模块
  • Wi-Fi component：Wi-Fi模块
  • LED
  • snap button：按钮

困难点

  • 体积小电池小导致待机时间短
  • 微型投影需要向水平方向投影，难以实现，投影效果马赛克级的分辨率
  • 手臂是弯曲平面，投影效果较差，需要对边缘作补偿，对瘦小的人显示效果更差
  • 对于有体毛的人，几乎无法投影完整画面，皮肤表面有太多障碍物
  • 触控的响应精确度和速度不高
