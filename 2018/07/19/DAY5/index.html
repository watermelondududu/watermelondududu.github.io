DAY5

querty键盘

  1. querty键盘缺点

    1. 被设计的初衷：防止滞黏，降低速度提高准确率
    2. 效率低
    3. 手部负担重
    4. 初学者学习成本高

  2. 其他键盘布局

    1. dovrak键盘
    2. malt键盘

  3. 但为什么仍是querty键盘被广泛使用

    1. 统一规范的问题
    2. 学习成本
    3. 没有有足够影响力的组织来推行（背后利益不够显著）
    4. 高速打字需求不明显

iPhoneX深度摄像头

  1. 原理及优劣势
  • 画出结构示意图
  • 深度成像有，iPhoneX选用：结构光
  • 结构光原理
  • 结构光优势和劣势
  • Face ID优势与劣势
  1. 基于深度摄像头的新颖的交互方式
    1. 保护隐私（监测到人像熄屏）
    2. 头动
    3. 眼动（自动浏览）
    4. 唇动（唇语识别）
    5. 表情（表情包工具）
    6. 距离（改变显示字体大小，有一个不变范围）
    7. 明暗
    8. 手势识别（针对特殊场景 比如做饭 无法触碰屏幕 ）

手机技术原理

005&006 传感器

  • 前置摄像头
  • 听筒
  • 距离传感器
  • 亮度传感器
  • 单摄
  • 双摄
    • 彩色+黑白（进光量高，分辨率高），夜晚成像效果增强
      • eg.华为p9
    • 广角+长焦（成本高）
    • 彩色（主）+副摄像头（常为深度摄像头）
    • 两个普通彩色摄像头（双目）
  • 三摄（双摄再加一）

屏幕

  • led
  • 电容屏
  • 柔性屏

屏下

  • 屏下摄像头
    • 钻孔摄像头
    • 透明度高的屏幕+算法补偿
  • 听筒、距离传感器：超声波传感器
  • 屏下指纹
    1. 光学方式，透过屏幕间隙
    2. 声波方式

产品设计进阶

  • 核心问题：开发::便于使用::的产品
  • 答卷时给自己的产品找亮点

kinect


  • 三只眼睛
    • 红外投影
    • 红外摄像头
    • 彩色摄像头
  • 四只耳朵：四元线性麦克风阵列
    • 声音从4个麦克风采集，内置数字信号处理DSP等组件，同时过滤背景噪声，可定位声源方向。

骨骼追踪

原理

  • rgb摄像头 -> 边缘检测
  • deepthcamara -> 人物背景分离
  • 提取20个关键节点 -> 骨骼模型

优点

  • 人物映射效果好
  • 多人识别
  • 全身体感交互

缺点

  • 延迟
  • 不能有障碍物
  • 需要较大场地0.5-4.5

身份识别

  • 图像识别（2D）

语音识别

麦克风

  • 辅助判断说话人位置
  • 降噪
