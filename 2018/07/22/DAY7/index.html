  • 输入
    • 语音
    • 物理按键

  • 输出

1.heyGoogle

::优点::

  • 拟人化
    • 亲切自然（打招呼）
    • 激发兴趣
    • 增加陪伴感
  • 误操作概率低、容错性高
  • 无需中断用户的当前行为

::缺点::

  • 被动式唤醒，主动提供服务？
  • 多人交互的场景的局限性
  • 距离限制

应用场景

  1. 做饭
  2. 瑜伽、运动时的提示
  3. 控制智能家电
  4. 提醒备忘
